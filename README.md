# Neo Mobility — Tesla Redesign (Next.js)

A concept redesign of the Tesla website — high-fidelity UI, cinematic imagery, and modern interactions powered by Next.js (App Router), Tailwind CSS, GSAP, and Framer Motion.  
This repository is a portfolio-quality demo showcasing UI/UX, animations, and interactive patterns for electric mobility products (Powerwall, vehicles, concept pages).

---

## ⚡ Quick links

- Live (example): `https://tesla-concept-redesigned.vercel.app/`

---

## ✨ Highlights / Stack

- **Framework**: Next.js 16 (App Router, Turbopack-ready)
- **Styling**: Tailwind CSS (utility-first)
- **Animations**: GSAP, Framer Motion, SplitType
- **Smooth scroll**: Lenis (plus optional locomotive-scroll)
- **Media**: webp images, Lottie animations, Sharp for image processing
- **Components**: Modular, domain-driven (homepage, blog, layout, ui)
- **Testing & linting**: `next lint` (basic setup)
